# Stock Price Prediction using Machine Learning 📈

In this project, I have utilized **Machine Learning** techniques to predict the stock prices of Powergrid. By analyzing historical data, the model forecasts future closing prices with high precision.

### 🚀 Features:
- **Dataset:** Historical Powergrid stock data (`powergrid.csv`).
- **Algorithm:** Linear Regression (Used for its efficiency in trend prediction).
- **Accuracy:** **99.45%** (Based on model evaluation metrics).

### 🛠️ Libraries Used:
- **Pandas:** For data manipulation and cleaning.
- **Scikit-learn:** For building and evaluating the Machine Learning model.
- **Matplotlib & Seaborn:** For data visualization and trend analysis.
